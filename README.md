## 목표기능
PLC에서 계측된 수질관리 데이터를 수집하여 수질계측값을 종합표로 표출하고 알람발생 조건에 맞게 알람을 발생 시킨다.
수질계측 값을 계통도에 따라 표출 시키며 변화에 따른 추이 그래프를 표시한다.
수질 분석 보고서 분석값 입력, 저장, 출력
현재 일시, 운전모드, 통신 상태, 수질계측값을 확인 할 수 있는 종합표를 구성한다.
